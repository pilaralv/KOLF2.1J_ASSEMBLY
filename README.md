# KOLF2.1J_ASSEMBLY
This is a repository for the code used for the KOLF2.1J iPSC genome assembly, annotation, and genomic analysis.

Link to manuscript: https://www.biorxiv.org/content/10.64898/2026.03.06.710144v2

KOLF Browser: https://genome.ucsc.edu/cgi-bin/hgGateway?hgHub_do_redirect=on&hgHubConnect.remakeTrackHub=on&hgHub_do_firstDb=1&hubUrl=https://research.nhgri.nih.gov/CustomTracks/T2T_hubs/T2T_test/KOLF2.1Jv1.1/hub.txt  

Main KOLF2.1J Github with data resources: https://github.com/marbl/KOLF2.1J-Diploid-Assembly?tab=readme-ov-file

Here, you will find the scripts that were used as part of the project. For data availability please see the Main KOLF 2.1J Github linked above.
